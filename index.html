📖 How to Use This App
🔄 Daily Workflow
1
Before going to site — Create a new Work Order with MO, PO, Area, Station.
2
At site (before work) — Open the Work Order → tap "📷 Before Work" → add photos. GPS is auto-tagged.
3
After work is done — Tap "📷 After Work" → add completion photos.
4
Mark as Bill Ready — Once all photos are uploaded, tap "✅ Mark Bill Ready".
5
File the bill — Go to 🧾 Bill Filing tab → click the work order → open Drive folder → upload to MSCB portal.
📂 How Drive is Organized
MSCB_FieldPhoto_Manager/
  └── [Area Name]/
      └── MO001_VillageGhatne/
          ├── 1_Before_Work/
          │    └── 2024-03-18_18.5204_73.8567_photo.jpg
          └── 2_After_Work/
              └── 2024-03-18_18.5205_73.8568_photo.jpg
Each filename contains: Date + GPS Coordinates + Type — so you always know where and when the photo was taken.

📄 PDF Bill Report
Open any Work Order → tap 📄 PDF Report button.
A PDF is generated with: MO/PO numbers, area, station, work type, date, contractor name, photo count, GPS coordinates, and Drive folder link — ready to attach to your bill.
